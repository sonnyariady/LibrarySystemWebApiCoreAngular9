The Simple Library System (Sistem Perpustakaan), contain:
- Backend: Web API .NET Core 3.1 (template from https://github.com/mahendraputra21/InventoryAPI/ which I modified by enabling CORS so that it can be access by other language, include angular)
- Frontend: Angular 9 CLI

1. Open DB folder for Generate SQL Script and Data To Your Local DB
2. Open Backend and open Solution for Backend
3. Open Frontend Folder (using Angular 9_
4. Run Solution from Backend
5. "npm install" on Frontend folder
6. "ng serve --o" on Frontend folder
